# css-queryKit
Simple include to allow added query functionality to css. 

## Project goals

- Allow front end devs comfortable with media queries to have more capabilities
- Allow for media query style datetime queries in css 
- TODO: Allow location based queries in css

## Getting Started

To use, simply include queryKit.php prior to any output. See demo files.
